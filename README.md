# xcx-checked
**小程序：view组件 单选操作，多选操作[可限制选中的数量]...**

小程序的组件checkbox，checkbox-group实现单选和多选功能是比较容易的，就是实际项目中考虑到页面美观和用户体验，还是选择自定义的，那么我们可以使用view组件，然后用js来处理实现跟表单的单选、多选一样的效果。
